# Link page (tipo LinkTree) - Dra. [NOME]

Página estática para centralizar links (WhatsApp, agendamento, Maps, Instagram, site).

## Editar links
Abra `links.js` e altere os itens do array `LINKS`.

## Rodar localmente
Abra `index.html` no navegador.

## Deploy (EasyPanel)
- Criar App -> Source: GitHub
- Build com Dockerfile
- Expor porta 80
